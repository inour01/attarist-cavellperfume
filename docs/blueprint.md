# **App Name**: Attarist E-Boutique

## Core Features:

- Homepage Hero: Implement a hero section with a full-screen parallax image, headline, and a call to action button linking to the product listing page.
- Product Listing with Faceted Search: Display products with a sidebar for filtering by olfactory family, notes, and format, enabling users to refine their search.
- Product Detail Page: Create a split-screen layout featuring a vertical image gallery and product details including title, price, 'Add to Cart' button, description, and an olfactory pyramid.
- Shopping Cart with Zustand: Implement a state-managed shopping cart using Zustand for tracking items added to the cart and managing the checkout process.
- Strictly Typed Mock Data: Generate a mock e-commerce dataset that simulates a call to the Medusa backend.
- Hazmat Warning Component: Implement a 'Ships via Ground Only (Precious Oils)' shipping restrictions notification that appears contextually.
- Top Seller Carousel: Showcase the available Top Seller in an auto-rotating carousel on the home page.

## Style Guidelines:

- Primary color: Deep Oud Brown (#3e2723) for a luxurious and earthy feel.
- Accent color: Muted Gold (#d4af37) to highlight elements and convey elegance.
- Background color: Cream/Off-White (#fdfbf7) to create a soft, organic backdrop.
- Text color: Charcoal Black (#1a1a1a) for clear and readable text.
- Headings: 'Playfair Display' or 'Cinzel' (serif) for an elegant and vintage feel.
- Body: 'Lato' or 'Inter' (sans-serif) for clean and modern readability.
- Use 'Lucide-React' or 'Heroicons' to implement modern icons. Ensure icons are visually simple and elegant.
- Employ generous negative space to achieve a minimalist design. Use high-resolution imagery to enhance visual appeal.
- Implement subtle animations, such as a slow parallax effect in the hero section and image swaps on product cards, to add depth and interactivity.
